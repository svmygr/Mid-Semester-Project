# Multi-Agent Research Assistant

## Overview
This project implements a simple multi-agent system where multiple AI agents collaborate to answer user questions.

## Agents

Web Agent
Searches the internet for information.

Document Agent
Retrieves information from PDF documents.

Summary Agent
Combines and summarizes results.

## Architecture

User Query
   ↓
Coordinator
   ↓
Multiple Agents
   ↓
Final Answer

## Installation

pip install -r requirements.txt

## Run

python main.py
